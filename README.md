# Handshakes DevOps Assignment

This project demonstrates Infrastructure as Code using CDK for Terraform (CDKTF)
with TypeScript.

## How to Run

```bash
npm install
cdktf synth

## Notes
- This project uses CDK for Terraform (CDKTF)
- No AWS credentials are required to run `cdktf synth`
- The command generates Terraform JSON files in the `cdktf.out/` directory
